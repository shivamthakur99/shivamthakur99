### 1. 🤖 Smart Form-to-Slack Automation Workflow
> **Tools:** n8n · Slack API · Conditional Logic · Webhooks

[![GitHub Repo](https://img.shields.io/badge/View_on_GitHub-100000?style=for-the-badge&logo=github&logoColor=white)](https://github.com/shivamthakur99/Smart-Form-to-Slack-Automation-Workflow-n8n)

An end-to-end automated workflow that eliminates manual monitoring for time-sensitive installation tracking.

**What it does:**
- Triggers on web form submissions and applies **date-based conditional logic** to detect installations within a 7-day window
- Integrates **Slack API** to push real-time alert messages to the relevant team channel when conditions are met
- Routes non-urgent submissions through a **fallback task-queue branch** for asynchronous review, ensuring zero lead drop-off
- Fully automated pipeline that improved team response time and operational efficiency

**Key Outcome:** Reduced manual monitoring effort significantly by automating time-sensitive installation tracking.
